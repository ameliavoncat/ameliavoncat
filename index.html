Trossello
[ App ] [ Github ]

  I was a major contributor to Trossello, an open source Trello clone built as an apprenticeship project at Learners Guild. The app was built using Node, Express, React, and SQL, styled with Sass, and tested using Mocha, Chai, and Selenium WebDriver.
  My contributions included key features such as commenting and labeling cards, reorganizing and refactoring the React portion of the app for consistency and modularity, implementing Selenium WebDriver for integration tests, and cross-browser compatibility of the drag and drop feature.
